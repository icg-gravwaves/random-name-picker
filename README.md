# Random Name Picker

Hosted at: [https://icg-gravwaves.github.io/random-name-picker/](https://icg-gravwaves.github.io/random-name-picker/)

A fun, interactive random name picker with multiple animation styles. Hosted entirely on GitHub Pages.

## Features

- **6 Animation Styles:**
  - 🎡 **Spin Wheel** — classic wheel with weighted slices
  - 🎰 **Slots** — stacked reel animation
  - 🪝 **Claw** — arcade claw with occasional fumbles
  - 🏁 **Race** — loser-mode race for tension
  - ⚔️ **Battle Arena** — multi-hit eliminations
  - 🔦 **Spotlight** — sweeping spotlight reveal

- **Controls & Options:**
  - Animation speed slider (0.25×–3×)
  - Optional weight toggles per name
  - Clear-all and sample-list loader

- **Sound & Celebration:**
  - Applause plays on winner reveal
  - Confetti burst on winner

- **Name Management:**
  - Add names, remove individually, clear all
  - Weights persist with names in localStorage

- **Responsive:**
  - Works on desktop, tablet, and mobile

## Quick Start

1. Clone or download this repository
2. Open `index.html` in your browser
3. Add names in the sidebar
4. Choose your animation style
5. Click "Pick Name!"

## Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to repository **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose **main** branch and **/ (root)** folder
5. Click **Save**

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `app.js` - Application logic and animations
